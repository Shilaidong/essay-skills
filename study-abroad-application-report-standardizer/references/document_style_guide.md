# Document Style Guide

## General design

A good consulting report should feel structured, spacious, and easy to scan.

Use:

- Clear title page.
- Executive summary near the front.
- Numbered headings.
- Short paragraphs.
- Tables for comparisons and checklists.
- Callout boxes for decisions, risks, and immediate actions.
- Gantt image for timeline.
- Consistent footnotes or source notes.

Avoid:

- Huge unbroken paragraphs.
- Overwide tables.
- Excessive colors.
- Raw Mermaid code in Word.
- Raw URLs scattered throughout body text unless requested.
- Dense bullet lists without hierarchy.

## Chinese report typography

Default:

- Page: A4.
- Margins: 2.0-2.3 cm.
- East Asian font: 宋体 / SimSun.
- Latin font: Times New Roman.
- Body: 10.5-11 pt.
- Line spacing: 1.25-1.45.
- H1: 16-18 pt bold.
- H2: 13-14 pt bold.
- H3: 11.5-12 pt bold.
- Tables: 8.5-10 pt depending on width.

Do not distribute font files.

## English report typography

Default:

- Body: 10.5-11 pt.
- Font: Times New Roman, Aptos, or requested brand font.
- Use clean heading hierarchy.

## Tables

Rules:

- Use compact tables only when they clarify comparison.
- Repeat header rows for long tables.
- Avoid more than 6-8 columns in portrait pages.
- Split wide matrices into multiple tables or use landscape orientation.
- Keep important school/program names unambiguous.

## Gantt charts

- Render as PNG.
- Use clear phase labels.
- Use milestone markers.
- Keep no more than 25-35 visible tasks on one chart.
- If timeline is too long, split into two charts.

## Source notes

Client-facing reports can use:

- Endnote table.
- Source appendix.
- Parenthetical source names.
- Short footnotes.

Keep internal URLs in a source ledger even when the client-facing report uses simplified notes.

## QA checklist

Before delivery:

- Cover page has correct student and date.
- No placeholder text remains.
- Page numbers if needed.
- Tables do not overflow.
- Chinese glyphs render correctly.
- Gantt image is readable.
- No broken hyperlinks if hyperlinks are included.
- Sources are readable.
- File name is professional.
