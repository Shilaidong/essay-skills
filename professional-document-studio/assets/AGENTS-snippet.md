# Professional document workflow

When a task requests a substantial report, proposal, strategy document, research brief, plan, case analysis, or polished Word artifact, use `$professional-document-studio`.

The user must supply domain-specific requirements in the task. Do not assume a fixed industry, topic, case module, number of options, or decision structure.

For DOCX output, always generate, render to page images, inspect every page, revise, and only then deliver the final file.
