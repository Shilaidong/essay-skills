# Workflow Checklist

## A. Before writing

- [ ] List all input files.
- [ ] Read the latest user-edited draft first to understand desired structure.
- [ ] Read all transcripts and score reports.
- [ ] Extract grade and teacher-comment evidence.
- [ ] Extract student/parent/advisor statements from interviews.
- [ ] Extract project timeline and existing competition/activity plan.
- [ ] Create `source_log.csv`.
- [ ] Create `activity_competition_map.md` with each core pathway, advancement mechanism, deliverables, risk, and fallback.
- [ ] Identify contradictions and missing facts.
- [ ] Verify current official policies online.

## B. Strategy design

- [ ] Determine application cycle.
- [ ] Determine major labels.
- [ ] Define Academic Spike / Impact Spike / Intellectual Vitality.
- [ ] Select 3 main reach schools when requested.
- [ ] Create extended school/program list.
- [ ] Write ED/EA/RD scenarios by score threshold.
- [ ] Write test plan by baseline and score goals.
- [ ] Write research/project plan.
- [ ] Write core activity and competition pathways, including advancement/selection mechanisms when relevant.
- [ ] Write leadership and essay narrative.

## C. Output build

- [ ] Draft Markdown.
- [ ] Render Gantt chart.
- [ ] Embed chart into Markdown.
- [ ] Convert/build Word document.
- [ ] Inspect tables and chart.
- [ ] Run quality check.
- [ ] Package outputs.
