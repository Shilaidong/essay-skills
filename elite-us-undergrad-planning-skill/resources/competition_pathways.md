# Competition Pathways Guide

This file is kept for backward compatibility. Use `core_activity_competition_pathways.md` as the primary reference.

Core rule: design applicant-specific core activities and competition pathways. Explain the advancement or selection mechanism if one exists. If there is no formal advancement, state that clearly and describe the evidence value: measurable impact, third-party validation, portfolio, publication, prototype, user data, recommendation material, or leadership proof.

Do not default every student to ISEF. Use ISEF/science-fair details only when the student’s target and official pathway genuinely require it.
