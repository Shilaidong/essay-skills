# Research Source Rules

Use official sources when current facts matter.

## Must verify

- standardized testing policies
- SAT/ACT dates and registration deadlines
- TOEFL/IELTS/Duolingo scoring updates
- school admissions rounds and restrictions
- school majors and departments
- financial aid policies
- competition/activity eligibility, materials, deadlines, team rules, judging criteria, and advancement mechanisms
- compliance rules for projects involving humans, minors, data privacy, biological materials, batteries/electrical systems, AI, hardware safety, or public-facing claims

## Do not rely on memory for

- whether a school is test-required or test-optional
- ED/REA restrictions
- current deadlines
- competition pathway details
- whether an activity has a formal advancement path
- scoring scale changes

## Write source summaries like this

- `ETS 官方说明：...`
- `College Board 官方日期页面显示：...`
- `Stanford 本科招生页面显示：...`
- `竞赛/活动官方规则显示：...`
- `官方晋级机制显示：...`

Keep raw URLs in appendix for planning documents. Use citations in chat responses when tools support citations.
