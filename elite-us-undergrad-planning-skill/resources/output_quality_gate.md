# Output Quality Gate

A report is not ready until it passes these checks:

## Content

- [ ] The first section gives a clear admissions-reader diagnosis.
- [ ] Current facts are detailed and source-grounded.
- [ ] Missing information is explicitly labeled.
- [ ] The student has a clear positioning sentence.
- [ ] Three core risks are concrete and actionable.
- [ ] School strategy includes exact programs, not just school names.
- [ ] Main 3 schools have trigger conditions when requested.
- [ ] SAT/TOEFL/ACT/IELTS plan is consistent with current baseline.
- [ ] Research/project plan includes data, controls/evaluation, outputs, and compliance.
- [ ] Core activity and competition pathways are truthful and conditional.
- [ ] Every competition/activity pathway states either its advancement mechanism or `无正式晋级机制`.
- [ ] Timeline has months and key weeks.

## Style

- [ ] Chinese is polished and consultant-grade.
- [ ] Tables are concise and not too wide.
- [ ] No generic “do research / do internship / volunteer more” language without outputs.
- [ ] No overpromising admission odds or advancement results.
- [ ] No unverified claims.

## Artifacts

- [ ] Markdown opens cleanly.
- [ ] Word opens cleanly.
- [ ] Word chart is an embedded image/chart.
- [ ] Image paths work in Markdown.
- [ ] File names are clean.
