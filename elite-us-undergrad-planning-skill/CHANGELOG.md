# Changelog

## v1.1.0

- Generalized the activity/competition planning module.
- Replaced ISEF-first language with `核心活动与竞赛路径`.
- Added explicit advancement/selection mechanism requirements for every competition or activity pathway.
- Added support for pathways with no formal advancement mechanism, emphasizing evidence value and measurable output.
- Added `resources/core_activity_competition_pathways.md`.
- Updated quality gate, checklist, example prompt, and Gantt example.
- Retained ISEF/science-fair handling only as an optional subpath when the student’s materials and goals justify it.
